# 锐动录屏SDK android版

全屏录制Android手机端桌面上的任意程序并生成视频文件或使用rtmp协议推流到云服务器；也支持OpenGLView、Unity3D或Coco2dx游戏引擎，并支持后台录制，为录制手游等手机端视频应用提供完美的解决方案。
